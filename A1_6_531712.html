Al imprimir informacin del dataframe se puede observar que la variable Calificacin es de tipo object y que tiene 2 registros no-nulos menos que la variable Tiempo, esto quiere decir que hay 2 registros que no tienen un valor vlido en la columna de Calificacin. Para corregir esto, se hace lo siguiente:

  • Se convierte toda la columna a numrica (float)
  • Se calcula la media de los registros que si tienen una calificacin vlida.
  • Se sustituyen los valores que no son vlidos por la media calculada.
  • Se eliminan registros residuales nulos que haya habido en la columna de tiempo
In[2]:
# Convertir la columna a numrica, forzando valores no numricos a NaN
df['Calificacion'] = pd.to_numeric(df['Calificacion'], errors='coerce')

# Calcular la media ignorando NaN
media = df['Calificacion'].mean()

# Reemplazar NaN con la media
df['Calificacion'] = df['Calificacion'].fillna(media)

from mlinsights.mlmodel import PiecewiseRegressor

ms = PiecewiseRegressor('bins').fit(Xq_train.iloc[:,1:],Y_train)

# Estimar el valor de Y en los datos de prueba

# Calcular RSS
RSS_ms = sum((list(Y_test)-yHat_ms) ** 2)

# Calcular RSE
RSE_ms = np.sqrt(RSS_ms / (len(Y_test) - 1))
